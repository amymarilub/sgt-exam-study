## üìñ How to Use

### Reading & Studying
1. Select a category from the first dropdown (e.g., "Collective Bargaining Agreement")
2. Select a specific document from the second dropdown (e.g., "Article 28 - Disciplinary Actions")
3. Read through the material
4. Use the text-to-speech feature to listen while multitasking
5. Highlight important sections using the color-coded toolbar

### Taking Quizzes
1. Go to **Quiz Mode**
2. Answer questions one at a time
3. Submit your answer to see if you're correct
4. Read the detailed explanation
5. Track your progress on the dashboard

### Highlighting Important Content
While reading, select text to bring up the highlight toolbar:
- **Yellow** - Important facts to remember
- **Red** - Exam traps and common mistakes
- **Green** - Must-know critical information
- **Blue** - Key procedures or processes

All highlights are saved and can be reviewed in **Review Mode**.

## üéì Study Tips

1. **Consistency over intensity** - Study 30-60 minutes daily rather than marathon sessions
2. **Use multiple modes** - Read, quiz, review highlights to reinforce learning
3. **Focus on critical content** - Articles 28, 29, and 12 are heavily tested
4. **Memorize numbers** - Timeframes, maximum values, and thresholds are common test questions
5. **Understand exam traps** - Pay special attention to content marked as "EXAM TRAPS"
6. **Practice scenarios** - Scenario-based questions test real-world application
